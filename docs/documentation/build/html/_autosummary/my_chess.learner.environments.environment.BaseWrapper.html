my_chess.learner.environments.environment.BaseWrapper¶

class my_chess.learner.environments.environment.BaseWrapper(env: AECEnv[AgentID, ObsType, ActionType])¶

Bases: BaseWrapper

To correct for bug in pettingzoo as of version 1.24.3. Issue is described here, https://github.com/Farama-Foundation/PettingZoo/issues/1176. Fix has been merged but not released. Until then, this patch suffices to ensure the agent selection actually transitions between players in the environment.

render() → None | np.ndarray | str | list¶

Renders the environment as specified by self.render_mode.

Render mode can be human to display a window. Other render modes in the default environments are ‘rgb_array’ which returns a numpy array and is supported by all environments outside of classic, and ‘ansi’ which returns the strings printed (specific to classic environments).
